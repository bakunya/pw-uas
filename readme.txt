REQUREIMENT
    - PHP 8.xx
    - MySQL / MariaDB
    - Apache2
    - Internet Connection

INSTALLATION
    - Adjust database configuration in ``application/config/database.php``
    - Change baseURL in ``application/config/config.php``
    - Import database file to ``localhost/phpmyadmin``
    - Open a browser then navigate to the URL according to the baseURL that has changed
